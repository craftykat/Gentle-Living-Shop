Our Old Website (built with WordPress.org)

This website is coded by us using HTML and CSS. But before we knew how to do that, we started with WordPress.org using the Twenty Twenty-One theme. It is a nice and simple theme, our favourite part is the dark mode. We wanted to build a website that is privacy respecting without having to display a cookie notice. The plugins we mention will therefore comply with GDPR policies.

WordPress plugins we liked using:

Watch the video: (coming soon!)

  • Polylang - for making the website multilingual. It is easy to use but it sets a cookie that can be hard to remove. You need to go into the plugin php file to comment out some parts.
  • Dinosaur Game - for the 404 page :)
  • Embed Privacy - prevent external embeds from loading without visiters' consent
  • Matomo Analytics - ethical analytics. It was fun to see where our visitors come from in the world but we decided to not use any analytics anymore on this current site because it doesn't really align with our beliefs of not being tracked on the web.
  • Menu Icons - fun icons for menus
  • Remove IP - a simple plugin that stops the recording of IPs from comments
  • WP Hide Login - change WP admin login url to prevent brute force attacks
  • WP Downgrade - rollback to old WP versions. This was especially helpful when WordPress updated to version 5.8 with new widget management which messed with our multilingual footer widgets. This plugin helped to revert WP back to the old version and restored the website.
  • WP 2FA - two-factor authentication for WP
  • randomPost - click to display a random post
  • WPFront Scroll Top - add a scroll to top button with ease
